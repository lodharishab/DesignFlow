
# DesignFlow - Project Overview (Version 0.01)

This document provides a summary of the DesignFlow project at its earliest stage.

## I. Core Structure & Public Pages

This version focuses on the public-facing brochure site.

*   **Homepage (`/`)**: Main landing page with a hero section.
*   **Services Landing Page (`/design-services`)**: A page to list services.
*   **All Services Page (`/services`)**: A catalog of all available design packages.
*   **Individual Service Detail Page (`/services/[serviceId]`)**: A detail view of a specific service.
*   **Portfolio Listing Page (`/portfolio`)**: A gallery showcasing completed projects.
*   **Blog Listing Page (`/blog`)**: Displays a grid of all published blog posts.
*   **Individual Blog Post Page (`/blog/[slug]`)**: Displays a single blog post.
*   **Layout Components**: A basic Navbar and Footer are in place.

## II. Technical Aspects

*   **Tech Stack**: Next.js (App Router), React, TypeScript, ShadCN UI, Tailwind CSS.
*   **Data Handling**: Uses mock data defined in TypeScript files for all dynamic content.
