# **App Name**: HYPE

## Core Features:

- Service Listing Management: Admins list and manage standard design services with fixed scopes and base prices.
- Designer Application & Approval: Freelancers apply to be sellers for specific services. Admin approves/rejects applications and assigns designers to service categories.
- Designer Dashboard: Designers manage orders, showcase portfolio for admin review and client credibility.
- Client Interface: Clients browse services, place orders, track progress, communicate with designers, and approve final designs.
- Admin Panel: Admins manage service listings, pricing, scopes, delivery timelines. Monitor order flow, performance analytics, and user activity.

## Style Guidelines:

- Soft color palettes to provide a clean and modern look.
- Clear typographic hierarchy for easy readability and navigation.
- Clean, minimal, and modern UI with a focus on ease of use for both clients and designers.
- Prioritize mobile-first design but ensure responsive web functionality.
- Use a consistent style of icons for navigation and service categories.
